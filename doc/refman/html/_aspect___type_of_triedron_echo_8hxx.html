enum Aspect_TypeOfTriedronEcho

Definition of the Triedron echo zone to highlight.

TOTE_NONE no echo zone TOTE_ORIGIN a box on origin of the Triedron TOTE_AXIS_X highlights the X axis TOTE_AXIS_Y highlights the Y axis TOTE_AXIS_Z highlights the Z axis TOTE_TEXT_X highlights the X character TOTE_TEXT_Y highlights the Y character TOTE_TEXT_Z highlights the Z character TOTE_01 to TOTE_10 not yet implemented.
