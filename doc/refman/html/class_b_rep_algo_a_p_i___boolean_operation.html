Detailed Description

The abstract class BooleanOperation is the root class of Boolean Operations (see Overview). Boolean Operations algorithm is divided onto two parts.

  • The first one is computing interference between arguments.
  • The second one is building the result of operation. The class BooleanOperation provides API level of both parts
