Detailed Description

This class implements the algorithms used to create 2d circles tangent to 3 points/lines/circles. The arguments of all construction methods are :

  • The three qualified elements for the tangency constraints (QualifiedCirc, QualifiedLine, Points).
  • A real Tolerance. Tolerance is only used in the limit cases. For example : We want to create a circle tangent to an UnqualifiedCirc C1 and an UnqualifiedCirc C2 and an UnqualifiedCirc C3 with a tolerance Tolerance. If we do not use Tolerance it is impossible to find a solution in the following case : C2 is inside C1 and there is no intersection point between the two circles, and C3 is completly outside C1. With Tolerance we will find a solution if the lowest distance between C1 and C2 is lower than or equal Tolerance.
