Detailed Description

Creates and updates a group of attributes for 3d line primitives. This group contains the colour, the type of line, and its thickness.

Colour : NOC_YELLOW Line type : TOL_SOLID Width : 1.0

Creates a context table for line primitives defined with the specified values. Warning: <AWidth> is the "linewidth scale factor". The nominal line width is 1 pixel.
