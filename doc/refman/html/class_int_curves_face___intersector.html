const Standard_Real  aTol,
const Standard_Boolean  aRestr = Standard_True 
)

Load a Face.

The Tolerance <Tol> is used to determine if the first point of the segment is near the face. In that case, the parameter of the intersection point on the line can be a negative value (greater than -Tol). If aRestr = true UV bounding box of face is used to restrict it's underlined surface, otherwise surface is not restricted
