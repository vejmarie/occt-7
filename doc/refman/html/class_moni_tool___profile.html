Detailed Description

A Profile gives access to a set of options :

  • it defines and gives access to options, by names specific to this Profile, defaulted to basic names of the Options
  • it records one or more Configurations, each of one is identified by a name, and commands for each option a switch value
  • a Configuration may be partial, i.e. concern only a subset of the complete list of options
  • Values are accessed through a Current Configuration, which name is fixed as ".", which can cumulate several configurations (especially when they are partial)

It may also bring a "fast" access to values, which has been formerly computed from current configuration or another given one. But its up-to-date status regarding existing configurations is not checked : it may be recomputed or cleared at any time. If not set, the current configuration is then taken
