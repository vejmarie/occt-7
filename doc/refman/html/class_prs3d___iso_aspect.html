Detailed Description

A framework to define the display attributes of isoparameters. This framework can be used to modify the default setting for isoparameters in Prs3d_Drawer.

Constructs a framework to define display attributes of isoparameters. These include:

  • the color attribute aColor
  • the type of line aType
  • the width value aWidth
  • aNumber, the number of isoparameters to be displayed.
