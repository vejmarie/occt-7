Detailed Description

defines an abstract owner of sensitive primitives. Owners are typically used to establish a connection between sensitive entities and high-level objects (e.g. presentations).

Priority : It's possible to give a priority: the scale : [0-9] ; the default priority is 0 it allows the predominance of one selected object upon another one if many objects are selected at the same time

example : Selection of shapes : the owners are selectable objects (presentations)

a user can give vertex priority [3], edges [2] faces [1] shape [0], so that if during selection one vertex one edge and one face are simultaneously detected, the vertex will only be hilighted.
