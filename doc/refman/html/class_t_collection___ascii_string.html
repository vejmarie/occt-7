TCollection_AsciiString Class Reference

A variable-length sequence of ASCII characters (normal 8-bit character type). It provides editing operations with built-in memory management to make AsciiString objects easier to use than ordinary character arrays. AsciiString objects follow value semantics; in other words, they are the actual strings, not handles to strings, and are copied through assignment. You may use HAsciiString objects to get handles to strings. More...

Detailed Description

A variable-length sequence of ASCII characters (normal 8-bit character type). It provides editing operations with built-in memory management to make AsciiString objects easier to use than ordinary character arrays. AsciiString objects follow value semantics; in other words, they are the actual strings, not handles to strings, and are copied through assignment. You may use HAsciiString objects to get handles to strings.
